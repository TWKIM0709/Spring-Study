   -SimpleConnectionMaker 클래스 생성
    (DB연결에 대한 함수)


   UserDao 가  SimpleConnectionMaker 의존 관계로 정의
   -N사 , D사 에는  UserDao 는  class 파일로
   -SimpleConnectionMaker java 제공 수정해서 사용해라 
   
      고객사
   UserDao > class 파일 형태로 ...
   SimpleConnectinMaker.java > 소스 제공 > 연결 코드 수정해서  사용
   
   UserDao 클래스와  SimpleConnectionMaker 클래스와의 관계
   
   
   
   
   1.상속(is ~a)
   2.포함 (has ~a) 
       의존(method 안에서)  : 함수 안에서 ......
       연관  - 복합(자동차는 엔진을 가지고 있다) , class Car {  new  Engine }
                   집합(학교는 학생을 가지고 있다)   , class School { Student st; add(Student st){ this.st  =st; } }                       
                          
	포함 - 복합연관
	class Car{
		Engine e;
		Car(){
			e = new Engine();
		}
	}
	포함 - 집합 연관
	class Car{
		Engine e;
		Car(Engine e){
			this.e = e;
		}
	}
